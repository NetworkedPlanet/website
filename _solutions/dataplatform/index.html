Networked Planet's Data Platform offers easy linked data publishing for businesses and organisations of any size. Using existing semantic web standards, the Data Platform makes it simple for both humans to browse hosted datasets, and for machines to easily read the data.

  • The Data Platform comes with a full website CMS, meaning that it is fully customisable with different themes and functionality such a blogs, additional information pages, plus a host of other available features.
  • Data is stored in an RDF Triple Store, upload data directly into your datasets using the simple web interface.
  • Publish numerous datasets, which each have their own homepage containing information about the dataset, links for download of the data and a query page with a user friendly query editor.
  • Datasets are housed within separate "stores", each store provides a homepage which can serve as a directory to the available datasets and has a unified SPARQL query endpoint which allows the execution of queries over all of the datasets in the store.

Key Features

  • Dataset descriptions automatically generated as RDF documents using the VoID vocabulary
  • SPARQL 1.1 query endpoints for datasets and multiple datasets within stores
  • N-Triples data dump exports
  • Import of RDF/XML, NT and TTL formatted data

Prerequisites

The Data Platform requires that you have a server operating system of at least Windows Server 2008 R2 or later.

Services

Networked Planet can provide:

  • Technical help when setting up your Data Platform
  • Analysis of your data
  • Extraction and conversion of your data into 5* linked data
