Linked Data

Linked Data is a technique for publishing data using existing web standards in a way that creates an interconnected Web of data. These relationships provide context and descriptive power to the core data making it more findable and increasing its usefulness to data consumers.

DataDock

Publishing Linked Open Data should not cost the Earth. We provide DataDock, a free service to convert and publish CSV data as Linked Open Data. Your data is stored in your own GitHub account along with a complete static data portal website. We provide the web magic that turns this static site into a Linked Open Data portal.
